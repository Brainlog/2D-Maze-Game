# COP-290 IIT Delhi Course Project

## About the Game :
This game is a 2D maze game based on map of IIT Delhi. The game uses SDL in C++ and socket.h for implementing multiplayer functionality.

## Dependencies :
LIBRARIES :- SDL2, SDL2_image, SDL2_ttf
